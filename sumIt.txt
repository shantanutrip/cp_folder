Calculate the sum of given numbers.

Input
An integer t, 1<=t<=100, denoting the number of testcases. Each testcase contains two lines, first denotes n, the number of inputs in th list. Next line contains n numbers.

Output
Sum of all the mumbers

Example
Sample input:

3
3
1 2 3
2
1 7
4
1 1 3 2

Sample output:

6
8
7
